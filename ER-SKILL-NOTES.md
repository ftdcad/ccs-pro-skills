# ER Skill — Key Learnings

## Executive Adjusters (e.g., Ben Kitzmiller / Steven Benjamin Kitzmiller)
- Still grade them — flag breaches, show findings — but note they're exempt from enforcement
- Exempt because they generate very few client complaints, not because rules don't apply
- Ben Kitzmiller goes by "Ben" — his email is ben@coastalclaims.net

## Adjuster Tiers (graded vs exempt)
- **Graded:** Apprentice adjusters, public adjusters, senior adjusters
- **Exempt from enforcement:** Executive adjusters (still grade, just note exemption)

## Referral Source Rules
- If adjuster IS the referral source on the same file: tracking only, no extra points/commission
- If adjuster is NOT the adjuster on the file but IS the referral source: earns 5 points
- Dual role (Adjuster + Referral Source) is normal — only flag if referral source shows actual commission/points
- 0 points on referral source = KPI tracking, no issue
- Actual points on referral source while also the adjuster = double commission, flag it

## 14-Day Communication Rule
- Applies to ALL claims regardless of phase (including Closed)
- A closed claim does NOT excuse the 14-day rule
- 14 days is 14 days. Period. No exceptions for claim status.
